== Description ==

This plugins helps you to create a Restaurant Menu from WooCommerce products. And generate a QR code for your menu. 



== Installation ==

How to install the plugin and get it working.


1. Upload `qreat-woocommerce-qr-menu.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is this plugin work without WooCommerce? =
No, This plugin only work If WooCommerce Installed
= Is there any limits =
No - You can add product, food and categories anything you want.

Please go to Products > Categories > Edit and set image for each category
